2.4 Image processing acceleration

resize​

Overview​

Implements image resizing. Only NV12 is supported for now.

Code repository: https://github.com/D-Robotics/hobot_cv

Supported platforms​

PlatformRuntime
RDK X3, RDK X3 ModuleUbuntu 20.04 (Foxy), Ubuntu 22.04 (Humble)
RDK X5, RDK X5 Module, RDK S100Ubuntu 22.04 (Humble)
RDK UltraUbuntu 20.04 (Foxy)
x86Ubuntu 20.04 (Foxy)

Prerequisites​

RDK​

  1. Flash Ubuntu 20.04 / 22.04 on the RDK.
  2. Install TogetheROS.Bot successfully.

x86​

  1. Ubuntu 20.04 is installed and TogetheROS.Bot is installed.

Usage​

RDK / x86​

  • Foxy
  • Humble
# Configure tros.b environment
source /opt/tros/setup.bash
# Configure tros.b environment
source /opt/tros/humble/setup.bash
# Copy models and config from the TogetheROS install path.
cp -r /opt/tros/${TROS_DISTRO}/lib/hobot_cv/config/ .

# Launch
ros2 launch hobot_cv hobot_cv_resize.launch.py

ProcessingFirst runSecond run
1920×1080 → 960×540297 ms15 ms

The first run configures VPS hardware and takes longer; subsequent runs reuse the configuration and are much faster.

color​

Overview​

Converts between NV12 and BGR24.

Code repository: https://github.com/D-Robotics/hobot_cv

Supported platforms​

PlatformRuntimeExample
RDK X5, RDK X5 Module, RDK S100Ubuntu 22.04 (Humble)NV12 ↔ BGR24 conversion

Prerequisites​

RDK​

  1. Ubuntu 22.04 image is flashed.
  2. TogetheROS.Bot is installed.

Usage​

RDK​

  • Humble
# Configure tros.b environment
source /opt/tros/humble/setup.bash
# Copy models and config from the tros.b install path.
cp -r /opt/tros/${TROS_DISTRO}/lib/hobot_cv/config/ .

# Launch
ros2 launch hobot_cv hobot_cv_conversion.launch.py
