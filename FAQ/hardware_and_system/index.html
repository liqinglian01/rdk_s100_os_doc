  2. жҹҘзңӢе·Іе®үиЈ…зҡ„ең°з“ңж ёеҝғеҠҹиғҪеҢ…зүҲжң¬пјҡ

    жҲ–иҖ…дҪҝз”Ё rdkos_info е‘Ҫд»ӨпјҲйҖӮз”ЁдәҺиҫғж–°зҡ„зі»з»ҹзүҲжң¬пјҢеҰӮ2.1.0еҸҠд»ҘеҗҺпјүпјҡ

Q3: дёҚеҗҢRDK OSзі»з»ҹзүҲжң¬е’ҢзЎ¬д»¶е№іеҸ°д№Ӣй—ҙжңүд»Җд№ҲеҜ№еә”е…ізі»пјҹвҖӢ

A:

  • RDK OS 2.x еҸҠжӣҙж–°зүҲжң¬зі»з»ҹ (еҰӮ2.0.0, 2.1.0, 3.0.x)пјҡ
    • еҹәдәҺD-Robotics LinuxејҖжәҗд»Јз ҒеҢ…еҲ¶дҪңгҖӮ
    • йҖҡеёёж”ҜжҢҒеҜ№еә”иҠҜзүҮзҡ„RDKзі»еҲ—зЎ¬д»¶пјҢдҫӢеҰӮRDK X3зҡ„2.x/3.xзі»з»ҹж”ҜжҢҒRDK X3гҖҒRDK X3 ModuleгҖӮ
  • RDK OS 1.x зүҲжң¬зі»з»ҹпјҡ
    • еҹәдәҺй—ӯжәҗLinuxзі»з»ҹеҲ¶дҪңпјҢеұһдәҺеҺҶеҸІзүҲжң¬гҖӮ
    • дё»иҰҒж”ҜжҢҒж—©жңҹзҡ„RDK X3зЎ¬д»¶гҖӮ

йҮҚиҰҒжіЁж„ҸдәӢйЎ№пјҡ

  • зүҲжң¬еҚҮзә§пјҡ 1.xзүҲжң¬зі»з»ҹж— жі•йҖҡиҝҮaptе‘Ҫд»ӨзӣҙжҺҘеҚҮзә§еҲ°2.xжҲ–жӣҙж–°зүҲжң¬зҡ„зі»з»ҹгҖӮеҰӮйңҖеҚҮзә§пјҢеҝ…йЎ»йҖҡиҝҮзғ§еҪ•ж–°зүҲжң¬зі»з»ҹй•ңеғҸзҡ„ж–№ејҸйҮҚж–°е®үиЈ…ж“ҚдҪңзі»з»ҹгҖӮ
  • TROSе…је®№жҖ§пјҡ дёҚеҗҢеӨ§зүҲжң¬зҡ„TROSпјҲеҰӮеҹәдәҺFoxyзҡ„TROSе’ҢеҹәдәҺHumbleзҡ„TROSпјүйҖҡеёёдёҺзү№е®ҡзҡ„RDK OSеӨ§зүҲжң¬з»‘е®ҡгҖӮдҫӢеҰӮпјҢRDK OS 2.x йҖҡеёёжҗӯиҪҪеҹәдәҺROS2 Foxyзҡ„TROSпјҢиҖҢRDK OS 3.x йҖҡеёёжҗӯиҪҪеҹәдәҺROS2 Humbleзҡ„TROSгҖӮ

Q4: ж‘„еғҸеӨҙжҸ’жӢ”жңүд»Җд№ҲжіЁж„ҸдәӢйЎ№пјҹвҖӢ

A: дёҘзҰҒеңЁејҖеҸ‘жқҝжңӘж–ӯз”өзҡ„жғ…еҶөдёӢжҸ’жӢ”ж‘„еғҸеӨҙпјҢеҗҰеҲҷйқһеёёе®№жҳ“зғ§еқҸж‘„еғҸеӨҙжЁЎз»„жҲ–дё»жқҝжҺҘеҸЈгҖӮ иҜ·еҠЎеҝ…еңЁж–ӯејҖејҖеҸ‘жқҝжүҖжңүз”өжәҗеҗҺпјҢеҶҚиҝӣиЎҢж‘„еғҸеӨҙзҡ„иҝһжҺҘжҲ–з§»йҷӨж“ҚдҪңгҖӮ

Q5: ејҖеҸ‘жқҝеҗҜеҠЁејӮеёёгҖҒдёҠз”өеҗҺж— д»»дҪ•жҳҫзӨәжҲ–еҸҚеӨҚйҮҚеҗҜпјҢеҸҜиғҪжҳҜд»Җд№ҲеҺҹеӣ пјҹеҰӮдҪ•жҺ’жҹҘпјҹвҖӢ

A: иҝҷзұ»й—®йўҳйҖҡеёёдёҺдҫӣз”өгҖҒеҗҜеҠЁд»ӢиҙЁпјҲSDеҚЎ/eMMCпјүжҲ–зЎ¬д»¶иҝһжҺҘжңүе…ігҖӮ

  • дҫӣз”өдёҚи¶іжҲ–дёҚзЁіе®ҡпјҡ

    • зҺ°иұЎпјҡ зі»з»ҹеңЁU-BootеҠ иҪҪеҶ…ж ёж—¶жҲ–еҶ…ж ёеҗҜеҠЁеҲқжңҹж— жҳҺжҳҫй”ҷиҜҜж—Ҙеҝ—е°ұзӣҙжҺҘйҮҚеҗҜпјӣз»ҝзҒҜзҠ¶жҖҒејӮеёёпјҲеҰӮRDK X3з»ҝзҒҜдёҚй—ӘзғҒжҲ–еёёдә®пјүпјӣHDMIе®Ңе…Ёй»‘еұҸгҖӮ
    • жҺ’жҹҘдёҺи§ЈеҶіпјҡ
      • зЎ®дҝқдҪҝз”Ёз¬ҰеҗҲејҖеҸ‘жқҝиҰҒжұӮзҡ„з”өжәҗйҖӮй…ҚеҷЁпјҲRDK X3жҺЁиҚҗиҮіе°‘5V/2AпјҢе»әи®®дҪҝз”Ёж”ҜжҢҒQC/PDзҡ„5V/3AжҲ–жӣҙй«ҳи§„ж јйҖӮй…ҚеҷЁпјүгҖӮ
      • зҰҒжӯўдҪҝз”ЁPCзҡ„USBжҺҘеҸЈдёәејҖеҸ‘жқҝдҫӣз”өгҖӮ
      • дҪҝз”ЁиҙЁйҮҸеҸҜйқ зҡ„USB Type-Cдҫӣз”өзәҝгҖӮ
      • еҸӮиҖғе®ҳж–№жҺЁиҚҗзҡ„еҹәзЎҖй…Қд»¶жё…еҚ•дёӯзҡ„з”өжәҗйҖӮй…ҚеҷЁеһӢеҸ·гҖӮ

  • еҗҜеҠЁд»ӢиҙЁй—®йўҳ (Micro SDеҚЎ/eMMC)пјҡ

    • зҺ°иұЎпјҡ дёІеҸЈж—Ҙеҝ—жҸҗзӨәж— жі•жҢӮиҪҪж–Үд»¶зі»з»ҹгҖҒжүҫдёҚеҲ°еҲҶеҢәгҖҒMMC/SDеҚЎеҲқе§ӢеҢ–й”ҷиҜҜжҲ–и¶…ж—¶гҖӮ
    • жҺ’жҹҘдёҺи§ЈеҶіпјҡ
      • зЎ®и®ӨSDеҚЎй•ңеғҸжҳҜеҗҰе·ІжӯЈзЎ®гҖҒе®Ңж•ҙең°зғ§еҪ•гҖӮ
      • е°қиҜ•йҮҚж–°зғ§еҪ•зі»з»ҹй•ңеғҸгҖӮ
      • жӣҙжҚўдёҖеј ж–°зҡ„гҖҒиҙЁйҮҸеҸҜйқ зҡ„й«ҳйҖҹMicro SDеҚЎгҖӮ
      • жё…жҙҒSDеҚЎж§Ҫе’ҢSDеҚЎйҮ‘жүӢжҢҮгҖӮ

  • дёІеҸЈиҜҜи§Ұиҝӣе…ҘU-Bootпјҡ

    • зҺ°иұЎпјҡ зі»з»ҹеҗҜеҠЁеҗҺеҒңз•ҷеңЁU-Bootе‘Ҫд»ӨиЎҢз•ҢйқўпјҲеҰӮ hobot>пјүгҖӮ
    • жҺ’жҹҘдёҺи§ЈеҶіпјҡ еҸҜиғҪжҳҜдёҠз”өж—¶и°ғиҜ•дёІеҸЈжңүйқһйў„жңҹиҫ“е…ҘгҖӮе°қиҜ•жӢ”жҺүдёІеҸЈзәҝеҗҺйҮҚж–°з»ҷи®ҫеӨҮдёҠз”өгҖӮеҰӮжһңеңЁU-Bootе‘Ҫд»ӨиЎҢпјҢеҸҜд»Ҙе°қиҜ•иҫ“е…Ҙ boot е‘Ҫд»Өе№¶еӣһиҪҰгҖӮ

  • е…¶д»–зЎ¬д»¶й—®йўҳжҲ–еӨ–и®ҫеҶІзӘҒпјҡ

    • еҰӮжһңд»ҘдёҠйғҪе·ІжҺ’йҷӨпјҢз§»йҷӨжүҖжңүйқһеҝ…иҰҒеӨ–и®ҫпјҲUSBи®ҫеӨҮгҖҒжү©еұ•жқҝзӯүпјүеҶҚе°қиҜ•еҗҜеҠЁгҖӮ
    • жһҒз«Ҝжғ…еҶөдёӢеҸҜиғҪеӯҳеңЁжқҝеҚЎжң¬иә«зЎ¬д»¶ж•…йҡңгҖӮ

  • иҜҰз»ҶжҺ’жҹҘжҢҮеҚ—пјҡ иҜ·еҸӮиҖғе®ҳж–№и®әеқӣзҡ„жқҝеҚЎж— жі•еҗҜеҠЁй—®йўҳе®ҡдҪҚжҢҮеҚ—гҖӮиҝһжҺҘи°ғиҜ•дёІеҸЈе№¶и®°еҪ•е®Ңж•ҙзҡ„еҗҜеҠЁж—Ҙеҝ—еҜ№дәҺй—®йўҳе®ҡдҪҚиҮіе…ійҮҚиҰҒгҖӮ

Q11: RDKејҖеҸ‘жқҝеҰӮдҪ•еҗҜз”Ёе’ҢдҪҝз”ЁADB (Android Debug Bridge) и°ғиҜ•еҠҹиғҪпјҹвҖӢ

  1. зЎ®и®ӨadbdжңҚеҠЎпјҡ жЈҖжҹҘжңҚеҠЎжҳҜеҗҰиҝҗиЎҢпјҢжҲ–жҳҜеҗҰжңүеҗҜеҠЁи„ҡжң¬гҖӮ
  2. USBжҺҘеҸЈжЁЎејҸпјҡ RDKжқҝеҚЎзҡ„USB Type-CеҸЈжҲ–зү№е®ҡзҡ„Micro USBеҸЈпјҲйҖҡеёёж ҮжңүOTGжҲ–DeviceеҠҹиғҪпјүеҸҜиғҪйңҖиҰҒиў«й…ҚзҪ®дёәUSB DeviceжЁЎејҸпјҲиҖҢдёҚжҳҜHostжЁЎејҸпјүжүҚиғҪиў«PCиҜҶеҲ«дёәADBи®ҫеӨҮгҖӮиҝҷжңүж—¶еҸҜд»ҘйҖҡиҝҮ srpi-config е·Ҙе…·жҲ–дҝ®ж”№и®ҫеӨҮж ‘/еҶ…ж ёеҸӮж•°жқҘй…ҚзҪ®гҖӮ
  3. PCз«ҜеҮҶеӨҮпјҡ еңЁжӮЁзҡ„з”өи„‘дёҠе®үиЈ…ADBе·Ҙе…·еҢ…пјҲйҖҡеёёдҪңдёәAndroid SDK Platform Toolsзҡ„дёҖйғЁеҲҶжҸҗдҫӣпјүгҖӮ
  4. иҝһжҺҘпјҡ дҪҝз”ЁUSBзәҝе°ҶPCдёҺRDKжқҝеҚЎдёҠй…ҚзҪ®дёәDeviceжЁЎејҸзҡ„USBеҸЈиҝһжҺҘгҖӮ
  5. йӘҢиҜҒпјҡ еңЁPCзҡ„е‘Ҫд»ӨиЎҢ/з»Ҳз«Ҝдёӯжү§иЎҢ adb devicesгҖӮеҰӮжһңдёҖеҲҮй…ҚзҪ®жӯЈзЎ®пјҢжӮЁеә”иҜҘиғҪзңӢеҲ°еҲ—еҮәзҡ„RDKи®ҫеӨҮгҖӮ
  6. дҪҝз”Ёпјҡ дёҖж—ҰиҝһжҺҘжҲҗеҠҹпјҢжӮЁе°ұеҸҜд»ҘдҪҝз”Ё adb shell и®ҝй—®жқҝеҚЎз»Ҳз«ҜпјҢadb push <жң¬ең°ж–Үд»¶> <жқҝеҚЎи·Ҝеҫ„> дёҠдј ж–Үд»¶пјҢadb pull <жқҝеҚЎж–Үд»¶> <жң¬ең°и·Ҝеҫ„> дёӢиҪҪж–Үд»¶зӯүгҖӮ

жіЁж„Ҹпјҡ е…·дҪ“зҡ„еҗҜз”ЁжӯҘйӘӨеҸҜиғҪеӣ RDKеһӢеҸ·е’Ңзі»з»ҹзүҲжң¬иҖҢејӮгҖӮиҜ·еҠЎеҝ…жҹҘйҳ…еҜ№еә”жқҝеҚЎе’Ңзі»з»ҹзүҲжң¬зҡ„е®ҳж–№ж–ҮжЎЈдёӯе…ідәҺADBеҠҹиғҪй…ҚзҪ®зҡ„иҜҰз»ҶиҜҙжҳҺгҖӮжңүж—¶пјҢе®ҳж–№жҸҗдҫӣзҡ„ bootloader жӣҙж–°ж•ҷзЁӢдёӯд№ҹеҸҜиғҪеҢ…еҗ«ADBзҡ„й…ҚзҪ®жҲ–дҪҝз”ЁеүҚжҸҗгҖӮ еҸӮиҖғ (жқҘиҮӘеҺҹе§Ӣж–ҮжЎЈпјҢеҸҜиғҪдё»иҰҒе…ідәҺbootloaderжӣҙж–°пјҢдҪҶеҸҜй—ҙжҺҘж¶үеҸҠADBзҺҜеўғ)пјҡbootloaderй•ңеғҸжӣҙж–° (е»әи®®жҹҘжүҫжӣҙдё“жіЁдәҺADBй…ҚзҪ®зҡ„е®ҳж–№ж–ҮжЎЈжҲ–зӨҫеҢәеё–еӯҗ)гҖӮ

Q12: ејҖеҸ‘жқҝе’Ңз”өи„‘д№Ӣй—ҙзҡ„ж–Үд»¶дј иҫ“жңүе“Әдәӣеёёз”Ёж–№ејҸпјҹвҖӢ

A: жңүеӨҡз§Қж–№ејҸеҸҜд»ҘеңЁRDKејҖеҸ‘жқҝе’Ңз”өи„‘д№Ӣй—ҙдј иҫ“ж–Үд»¶пјҡ

  1. SCP (Secure Copy Protocol) - еҹәдәҺSSHпјҡ

    • д»Һз”өи„‘жӢ·иҙқж–Үд»¶еҲ°ејҖеҸ‘жқҝпјҡ 
      # жӢ·иҙқеҚ•дёӘж–Үд»¶
      scp /path/to/local_file sunrise@<ејҖеҸ‘жқҝIPең°еқҖ>:/path/on/rdk/
      # жӢ·иҙқж•ҙдёӘж–Үд»¶еӨ№ (дҪҝз”Ё -r йҖүйЎ№)
      scp -r /path/to/local_folder sunrise@<ејҖеҸ‘жқҝIPең°еқҖ>:/path/on/rdk/
    • д»ҺејҖеҸ‘жқҝжӢ·иҙқж–Үд»¶еҲ°з”өи„‘пјҡ 
      scp sunrise@<ејҖеҸ‘жқҝIPең°еқҖ>:/path/on/rdk/remote_file /path/to/local_destination/
      scp -r sunrise@<ејҖеҸ‘жқҝIPең°еқҖ>:/path/on/rdk/remote_folder /path/to/local_destination/

    йңҖиҰҒз”өи„‘з«ҜжңүSCPе®ўжҲ·з«ҜпјҲLinux/macOSиҮӘеёҰпјҢWindowsеҸҜз”ЁWinSCPгҖҒMobaXtermжҲ–Git Bashдёӯзҡ„scpпјүгҖӮ

  2. SFTP (SSH File Transfer Protocol) - еҹәдәҺSSHпјҡ и®ёеӨҡFTPе®ўжҲ·з«ҜиҪҜд»¶пјҲеҰӮFileZilla, WinSCPпјүж”ҜжҢҒSFTPеҚҸи®®пјҢеҸҜд»ҘжҸҗдҫӣеӣҫеҪўеҢ–зҡ„ж–Үд»¶дј иҫ“з•ҢйқўгҖӮиҝһжҺҘж—¶йҖүжӢ©SFTPеҚҸи®®пјҢе№¶дҪҝз”ЁSSHзҡ„з”ЁжҲ·еҗҚгҖҒеҜҶз Ғе’ҢIPең°еқҖгҖӮ

  3. USBеӯҳеӮЁи®ҫеӨҮ (Uзӣҳ/з§»еҠЁзЎ¬зӣҳ)пјҡ

    • е°ҶUзӣҳзӯүж јејҸеҢ–дёәејҖеҸ‘жқҝж”ҜжҢҒзҡ„ж–Үд»¶зі»з»ҹпјҲеҰӮFAT32, exFAT, жҲ–е®үиЈ…дәҶntfs-3gеҗҺзҡ„NTFSпјүгҖӮ
    • еңЁз”өи„‘дёҠеӯҳе…Ҙж–Үд»¶пјҢз„¶еҗҺе°ҶUзӣҳжҸ’е…ҘејҖеҸ‘жқҝзҡ„USB HostеҸЈгҖӮ
    • еңЁејҖеҸ‘жқҝдёҠжҢӮиҪҪUзӣҳ (sudo mount /dev/sda1 /mnt/usb_disk - и®ҫеӨҮиҠӮзӮ№еҸҜиғҪеҸҳеҢ–)пјҢз„¶еҗҺиҝӣиЎҢж–Үд»¶ж“ҚдҪңгҖӮ

  4. ADB (Android Debug Bridge) - еҰӮжһңе·Ій…ҚзҪ®е№¶иҝһжҺҘпјҡ

    • д»Һз”өи„‘жҺЁйҖҒж–Үд»¶еҲ°ејҖеҸ‘жқҝпјҡ 
      adb push C:\local\path\file.txt /remote/path/on/rdk/
    • д»ҺејҖеҸ‘жқҝжӢүеҸ–ж–Үд»¶еҲ°з”өи„‘пјҡ 
      adb pull /remote/path/on/rdk/file.txt C:\local\path\

  5. зҪ‘з»ңе…ұдә«жңҚеҠЎ (еҰӮSamba, NFS)пјҡ еҸҜд»ҘеңЁејҖеҸ‘жқҝдёҠй…ҚзҪ®SambaжҲ–NFSжңҚеҠЎпјҢе°Ҷзү№е®ҡзӣ®еҪ•е…ұдә«еҲ°еұҖеҹҹзҪ‘пјҢз„¶еҗҺеңЁз”өи„‘дёҠеғҸи®ҝй—®зҪ‘з»ңй©ұеҠЁеҷЁдёҖж ·и®ҝй—®иҝҷдәӣж–Үд»¶гҖӮй…ҚзҪ®зӣёеҜ№еӨҚжқӮдёҖдәӣгҖӮ

  6. Python HTTPжңҚеҠЎеҷЁ (дёҙж—¶е°Ҹж–Үд»¶е…ұдә«)пјҡ еҰӮжһңеңЁејҖеҸ‘жқҝзҡ„жҹҗдёӘзӣ®еҪ•дёӢжңүжғіи®©з”өи„‘дёӢиҪҪзҡ„ж–Үд»¶пјҢеҸҜд»ҘеңЁиҜҘзӣ®еҪ•дёӢеҝ«йҖҹеҗҜеҠЁдёҖдёӘHTTPжңҚеҠЎеҷЁпјҡ

    # еңЁејҖеҸ‘жқҝдёҠпјҢиҝӣе…ҘиҰҒе…ұдә«зҡ„зӣ®еҪ•
    cd /path/to/share
    python3 -m http.server 8000

    з„¶еҗҺеңЁз”өи„‘зҡ„жөҸи§ҲеҷЁдёӯи®ҝй—® http://<ејҖеҸ‘жқҝIPең°еқҖ>:8000 еҚіеҸҜзңӢеҲ°ж–Үд»¶еҲ—иЎЁе№¶дёӢиҪҪгҖӮ

йҖүжӢ©е“Әз§Қж–№ејҸеҸ–еҶідәҺж–Үд»¶еӨ§е°ҸгҖҒдј иҫ“йў‘зҺҮгҖҒзҪ‘з»ңзҺҜеўғд»ҘеҸҠдёӘдәәеҒҸеҘҪгҖӮеҜ№дәҺејҖеҸ‘иҝҮзЁӢдёӯзҡ„д»Јз Ғе’Ңй…ҚзҪ®ж–Үд»¶еҗҢжӯҘпјҢSCP/SFTPжҲ–VS Codeзҡ„Remote-SSHеҶ…зҪ®зҡ„ж–Үд»¶еҗҢжӯҘеҠҹиғҪйҖҡеёёжңҖдёәдҫҝжҚ·гҖӮ

Q13: : зі»з»ҹжү§иЎҢ apt upgrade ж—¶жЎҢйқўзҺҜеўғй»‘еұҸжҖҺд№ҲеҠһпјҹвҖӢ

A: е»әи®®йҖҡиҝҮдёІеҸЈз»Ҳз«ҜжҲ–SSHиҝңзЁӢзҷ»еҪ•еҗҺпјҢеңЁзәҜеӯ—з¬Ұз•Ңйқўзҡ„з»Ҳз«Ҝдёӯжү§иЎҢзі»з»ҹжӣҙж–°е‘Ҫд»ӨпјҲеҰӮ sudo apt update && sudo apt upgradeпјүгҖӮзӣҙжҺҘеңЁеӣҫеҪўжЎҢйқўзҡ„з»Ҳз«Ҝдёӯжӣҙж–°пјҢжңүж—¶еңЁеҚҮзә§жЎҢйқўиҮӘиә«зӣёе…ізҡ„еҢ…ж—¶еҸҜиғҪдјҡеҜјиҮҙжҳҫзӨәдёӯж–ӯжҲ–X ServerйҮҚеҗҜпјҢд»ҺиҖҢйҖ жҲҗй»‘еұҸзҺ°иұЎгҖӮ

Q14: дёәд»Җд№ҲSDеҚЎжҸ’дёҠеҺ»иғҪеҗҜеҠЁпјҢдҪҶжҳҜжӢ”дёӢжқҘеҗҺдёӢж¬Ўе°ұеҗҜеҠЁдёҚдәҶдәҶпјҹвҖӢ

иҜ·зЎ®и®ӨжӮЁзҡ„жқҝеҚЎеһӢеҸ·е’ҢеҪ“еүҚзҡ„еҗҜеҠЁд»ӢиҙЁи®ҫзҪ®гҖӮ

Q15: RDK OSзҡ„ServerзүҲиғҪзӣҙжҺҘеҚҮзә§дёәDesktopзүҲеҗ—пјҹвҖӢ

A: RDK OSзҡ„ServerзүҲе’ҢDesktopзүҲеңЁйў„иЈ…зҡ„иҪҜд»¶еҢ…дёҠеӯҳеңЁжҳҫи‘—е·®ејӮпјҢжңҖдё»иҰҒзҡ„жҳҜDesktopзүҲеҢ…еҗ«дәҶеӣҫеҪўз”ЁжҲ·з•ҢйқўпјҲеҰӮXFCEжЎҢйқўзҺҜеўғпјүеҸҠе…¶зӣёе…із»„д»¶пјҢиҖҢServerзүҲйҖҡеёёдёҚеҢ…еҗ«иҝҷдәӣпјҢд»ҘиҠӮзңҒиө„жәҗгҖӮ

  • зҗҶи®әдёҠпјҡ жӮЁеҸҜд»ҘйҖҡиҝҮеңЁServerзүҲзі»з»ҹдёҠжүӢеҠЁе®үиЈ…жүҖжңүDesktopзүҲжүҖйңҖзҡ„иҪҜд»¶еҢ…пјҲеҰӮxserver-xorg, xfce4, lightdmзӯүд»ҘеҸҠе®ғд»¬зҡ„дҫқиө–пјүжқҘе°Ҷе…¶вҖңеҚҮзә§вҖқдёәдёҖдёӘе…·жңүеӣҫеҪўз•Ңйқўзҡ„зі»з»ҹгҖӮ
  • е®ҳж–№ж”ҜжҢҒдёҺзЁіе®ҡжҖ§пјҡ е®ҳж–№йҖҡеёёдёҚжҸҗдҫӣжҲ–дёҚжҺЁиҚҗиҝҷз§ҚжүӢеҠЁеҚҮзә§и·Ҝеҫ„пјҢе№¶дё”дёҚеҜ№йҖҡиҝҮжӯӨж–№ејҸжһ„е»әзҡ„Desktopзі»з»ҹзҡ„зЁіе®ҡжҖ§е’Ңе®Ңж•ҙжҖ§иҝӣиЎҢдҝқиҜҒжҲ–жөӢиҜ•гҖӮжүӢеҠЁе®үиЈ…иҝҮзЁӢеӨҚжқӮпјҢе®№жҳ“йҒ—жјҸдҫқиө–жҲ–дә§з”ҹй…ҚзҪ®еҶІзӘҒгҖӮ
  • жҺЁиҚҗеҒҡжі•пјҡ еҰӮжһңжӮЁйңҖиҰҒDesktopзүҲзҡ„е®Ңж•ҙеҠҹиғҪе’ҢжңҖдҪідҪ“йӘҢпјҢејәзғҲе»әи®®зӣҙжҺҘдёӢиҪҪе№¶зғ§еҪ•е®ҳж–№еҸ‘еёғзҡ„DesktopзүҲжң¬зі»з»ҹй•ңеғҸгҖӮиҝҷжҳҜзЎ®дҝқзі»з»ҹзЁіе®ҡжҖ§е’ҢеҠҹиғҪе®Ңж•ҙзҡ„жңҖдҪійҖ”еҫ„гҖӮ

Q16: дёәд»Җд№ҲиҝһжҺҘHDMIжҳҫзӨәеҷЁеҗҺжІЎжңүз”»йқўжҳҫзӨәпјҢжҲ–иҖ…жҳҫзӨәејӮеёёпјҹвҖӢ

Q17: иҝһжҺҘзҡ„HDMIжҳҫзӨәеҷЁдёҚиў«ж”ҜжҢҒпјҢеҰӮдҪ•жҠ“еҸ–EDIDдҝЎжҒҜд»ҘдҫӣжҠҖжңҜж”ҜжҢҒеҲҶжһҗпјҹвҖӢ

  3. еңЁPCдёҠиҜ»еҸ–пјҡ еҰӮжһңиҜҘжҳҫзӨәеҷЁиҝһжҺҘеҲ°дёҖеҸ°иЈ…жңүLinuxжҲ–Windowsзҡ„PCдёҠеҸҜд»ҘжӯЈеёёе·ҘдҪңпјҢд№ҹеҸҜд»Ҙе°қиҜ•еңЁPCдёҠиҜ»еҸ–е…¶EDIDдҝЎжҒҜгҖӮ

Q18:зі»з»ҹеҗҜеҠЁзҡ„ж—¶еҖҷеҚЎеңЁ hobot> U-Bootе‘Ҫд»ӨиЎҢз•ҢйқўжҖҺд№ҲеҠһпјҹвҖӢ

  1. дёІеҸЈе№Іжү°пјҡ еңЁжқҝеҚЎдёҠз”өеҗҜеҠЁзҡ„жңҖеҲқеҮ з§’еҶ…пјҢеҰӮжһңи°ғиҜ•дёІеҸЈжҺҘж”¶еҲ°дәҶжҹҗдәӣйқһйў„жңҹзҡ„еӯ—з¬ҰжҲ–з”өе№ідҝЎеҸ·пјҲдҫӢеҰӮпјҢй”®зӣҳиҜҜи§ҰгҖҒдёІеҸЈз»Ҳз«ҜиҪҜд»¶иҮӘеҠЁеҸ‘йҖҒзҡ„жҹҗдәӣжҺ§еҲ¶еӯ—з¬ҰпјүпјҢеҸҜиғҪдјҡдёӯж–ӯU-Bootзҡ„иҮӘеҠЁеј•еҜјжөҒзЁӢпјҢдҪҝе…¶еҒңеңЁе‘Ҫд»ӨиЎҢгҖӮ

  3. еј•еҜји„ҡжң¬й—®йўҳпјҡ U-Bootжү§иЎҢзҡ„еј•еҜји„ҡжң¬пјҲboot scriptпјүеҰӮжһңеӯҳеңЁй”ҷиҜҜжҲ–иў«дёӯж–ӯгҖӮ

    • дҪҝз”Ёзі»з»ҹе‘Ҫд»ӨпјҲеҰӮ hrut_somstatusпјҢжҲ–иҜ»еҸ– /sys/class/thermal/thermal_zoneX/temp ж–Үд»¶еҶ…е®№пјүжқҘе®һж—¶зӣ‘жҺ§иҠҜзүҮжё©еәҰгҖӮ
    • дәҶи§ЈжқҝеҚЎиҠҜзүҮзҡ„е®үе…Ёе·ҘдҪңжё©еәҰиҢғеӣҙпјҢйҒҝе…Қй•ҝж—¶й—ҙи¶…еҮәдёҠйҷҗгҖӮ
  4. дјҳеҢ–еә”з”ЁиҙҹиҪҪпјҡ
    • еҰӮжһңеҸҜиғҪпјҢдјҳеҢ–жӮЁзҡ„еә”з”ЁзЁӢеәҸпјҢеҮҸе°‘дёҚеҝ…иҰҒзҡ„и®Ўз®—пјҢйҷҚдҪҺCPU/BPUзҡ„жҢҒз»ӯй«ҳиҙҹиҪҪгҖӮ
    • иҖғиҷ‘жҳҜеҗҰеҸҜд»ҘйҖҡиҝҮз®—жі•дјҳеҢ–гҖҒжЁЎеһӢиҪ»йҮҸеҢ–зӯүж–№ејҸйҷҚдҪҺеҠҹиҖ—е’ҢеҸ‘зғӯгҖӮ
  5. жЈҖжҹҘдҫӣз”өпјҡ иҷҪз„¶дёҚзӣҙжҺҘеҜјиҮҙеҸ‘зғӯпјҢдҪҶдёҚзЁіе®ҡзҡ„дҫӣз”өеҸҜиғҪеҜјиҮҙиҠҜзүҮе·ҘдҪңејӮеёёпјҢй—ҙжҺҘеҪұе“Қжё©еәҰгҖӮ

йҮҚиҰҒжҸҗзӨәпјҡ вҖңеҸ‘зғӯйҮҸе°ҸдёҚд»ЈиЎЁжё©еәҰдјҡдҪҺвҖқгҖӮеҚідҪҝиҠҜзүҮжң¬иә«и®ҫи®ЎеҠҹиҖ—дёҚй«ҳпјҢеҰӮжһңж•ЈзғӯдёҚиүҜпјҢзғӯйҮҸз§ҜиҒҡд»Қз„¶дјҡеҜјиҮҙиЎЁйқўжё©еәҰеҝ«йҖҹеҚҮй«ҳгҖӮиүҜеҘҪзҡ„ж•Јзғӯи®ҫи®ЎжҳҜдҝқиҜҒеөҢе…ҘејҸзі»з»ҹзЁіе®ҡиҝҗиЎҢзҡ„е…ій”®гҖӮ

Q21: еҰӮдҪ•еңЁCondaиҷҡжӢҹзҺҜеўғдёӯиҺ·еҸ–е’ҢдҪҝз”Ёең°з“ңжңәеҷЁдәәRDKзү№е®ҡзҡ„PythonеҢ…пјҲеҰӮ hobot.GPIO, hobot_dnn зӯүпјүпјҹвҖӢ

A: ең°з“ңжңәеҷЁдәәе®ҳж–№жҸҗдҫӣзҡ„ hobot.GPIOгҖҒhobot_dnn зӯүPythonеҢ…йҖҡеёёжҳҜдёәRDKзҡ„зі»з»ҹPythonзҺҜеўғйў„зј–иҜ‘е’ҢдјҳеҢ–зҡ„пјҢе®ғд»¬еҸҜиғҪдҫқиө–дәҺзі»з»ҹеә•еұӮзҡ„зү№е®ҡеә“ж–Үд»¶е’Ңй©ұеҠЁзЁӢеәҸгҖӮеңЁCondaзӯүPythonиҷҡжӢҹзҺҜеўғдёӯдҪҝз”ЁиҝҷдәӣеҢ…еҸҜиғҪдјҡйҒҮеҲ°дёҖдәӣжҢ‘жҲҳпјҢеӣ дёәиҷҡжӢҹзҺҜеўғж—ЁеңЁйҡ”зҰ»дҫқиө–гҖӮ

д»ҘдёӢжҳҜдёҖдәӣеҸҜиғҪзҡ„ж–№жі•е’ҢжіЁж„ҸдәӢйЎ№пјҡ

  1. е®ҳж–№жҳҜеҗҰжҸҗдҫӣCondaж”ҜжҢҒжҲ– .whl ж–Үд»¶пјҡ
    • йҰ–е…ҲпјҢжҹҘйҳ…жңҖж–°зҡ„ең°з“ңжңәеҷЁдәәе®ҳж–№ж–ҮжЎЈгҖҒејҖеҸ‘иҖ…зӨҫеҢәжҲ–GitHubд»“еә“пјҢзңӢе®ҳж–№жҳҜеҗҰжҸҗдҫӣдәҶй’ҲеҜ№CondaзҺҜеўғзҡ„е®үиЈ…иҜҙжҳҺпјҢжҲ–иҖ…жҳҜеҗҰзӣҙжҺҘеҸ‘еёғдәҶеҸҜд»ҘеңЁCondaзҺҜеўғдёӯйҖҡиҝҮ pip е®үиЈ…зҡ„ .whl ж јејҸзҡ„иҝҷдәӣеҢ…гҖӮиҝҷжҳҜжңҖзҗҶжғізҡ„жғ…еҶөгҖӮ
  2. е°қиҜ•еңЁCondaзҺҜеўғдёӯйҖҡиҝҮ pip е®үиЈ…зі»з»ҹи·Ҝеҫ„дёӢзҡ„еҢ…пјҲеҰӮжһң .whl дёҚеҸҜз”Ёпјүпјҡ
    • еҰӮжһңиҝҷдәӣеҢ…е·Із»Ҹе®үиЈ…еңЁRDKзҡ„зі»з»ҹPythonзҺҜеўғдёӯпјҲдҫӢеҰӮеңЁ /usr/lib/python3/dist-packages/ жҲ–зұ»дјји·Ҝеҫ„дёӢпјүпјҢе№¶дё”жӮЁзҡ„CondaзҺҜеўғдҪҝз”Ёзҡ„PythonзүҲжң¬дёҺзі»з»ҹPythonеҸҠиҝҷдәӣеҢ…зј–иҜ‘ж—¶жүҖз”Ёзҡ„PythonзүҲжң¬е…је®№пјҢжңүж—¶еҸҜд»ҘзӣҙжҺҘеңЁжҝҖжҙ»CondaзҺҜеўғеҗҺпјҢе°қиҜ•з”Ё pip жҢҮеҗ‘иҝҷдәӣеҢ…зҡ„и·Ҝеҫ„иҝӣиЎҢе®үиЈ…пјҢдҪҶиҝҷйҖҡеёёдёҚиў«жҺЁиҚҗпјҢдё”жҲҗеҠҹзҺҮдёҚй«ҳпјҢеӣ дёә pip дё»иҰҒз”ЁдәҺд»ҺPyPIжҲ–жң¬ең°.whl/жәҗз ҒеҢ…е®үиЈ…гҖӮ
  3. дҝ®ж”№ PYTHONPATH жҲ– sys.path (дёҚжҺЁиҚҗпјҢжҳ“еҮәй”ҷ)пјҡ
    • дёҖз§ҚдёҚи§„иҢғзҡ„ж–№жі•жҳҜпјҢеңЁжҝҖжҙ»CondaзҺҜеўғеҗҺпјҢжүӢеҠЁе°Ҷзі»з»ҹPythonзҺҜеўғдёӯиҝҷдәӣзү№е®ҡеҢ…зҡ„и·Ҝеҫ„ж·»еҠ еҲ°CondaзҺҜеўғзҡ„ PYTHONPATH зҺҜеўғеҸҳйҮҸдёӯпјҢжҲ–иҖ…еңЁPythonи„ҡжң¬дёӯеҠЁжҖҒдҝ®ж”№ sys.pathгҖӮ
    • йЈҺйҷ©пјҡ иҝҷз§Қж–№жі•йқһеёёе®№жҳ“еҜјиҮҙдҫқиө–еҶІзӘҒгҖҒзүҲжң¬дёҚеҢ№й…Қд»ҘеҸҠйҡҫд»ҘиҝҪиёӘзҡ„иҝҗиЎҢж—¶й”ҷиҜҜпјҢеӣ дёәCondaзҺҜеўғзҡ„йҡ”зҰ»жҖ§иў«з ҙеқҸдәҶгҖӮејәзғҲдёҚжҺЁиҚҗз”ЁдәҺз”ҹдә§жҲ–еӨҚжқӮйЎ№зӣ®гҖӮ
  4. дҪҝз”Ёзі»з»ҹPythonзҺҜеўғпјҡ
    • еҰӮжһңжӮЁзҡ„йЎ№зӣ®еҜ№PythonзҺҜеўғйҡ”зҰ»зҡ„иҰҒжұӮдёҚжҳҜйқһеёёдёҘж јпјҢжҲ–иҖ…дё»иҰҒе°ұжҳҜеӣҙз»•иҝҷдәӣRDKзү№е®ҡеҢ…иҝӣиЎҢејҖеҸ‘пјҢжңҖз®ҖеҚ•зӣҙжҺҘзҡ„ж–№жі•еҸҜиғҪе°ұжҳҜзӣҙжҺҘдҪҝз”ЁRDKзі»з»ҹиҮӘеёҰзҡ„PythonзҺҜеўғпјҢиҖҢдёҚжҳҜCondaгҖӮиҝҷдәӣеҢ…еңЁзі»з»ҹзҺҜеўғдёӯйҖҡеёёжҳҜй…ҚзҪ®еҘҪзҡ„гҖӮ
  5. е®№еҷЁеҢ–ж–№жЎҲ (Docker)пјҡ
    • еҰӮжһңең°з“ңжңәеҷЁдәәе®ҳж–№жҸҗдҫӣдәҶеҢ…еҗ«иҝҷдәӣеҢ…е’Ңе®Ңж•ҙдҫқиө–зҡ„Dockerй•ңеғҸпјҢйӮЈд№ҲеңЁDockerе®№еҷЁдёӯдҪҝз”ЁиҝҷдәӣеҠҹиғҪжҳҜжӣҙеҸҜйқ зҡ„йҡ”зҰ»е’ҢйғЁзҪІж–№жЎҲгҖӮ
  6. д»Һжәҗз Ғзј–иҜ‘пјҲеҰӮжһңе®ҳж–№жҸҗдҫӣжәҗз Ғдё”е…Ғи®ёпјүпјҡ
    • еҰӮжһңиҝҷдәӣзү№е®ҡеҢ…зҡ„жәҗз ҒжҳҜејҖж”ҫзҡ„пјҢе№¶дё”жңүй’ҲеҜ№ARMжһ¶жһ„зҡ„зј–иҜ‘жҢҮеҚ—пјҢзҗҶи®әдёҠжӮЁеҸҜд»Ҙе°қиҜ•еңЁжӮЁзҡ„CondaзҺҜеўғдёӯд»Һжәҗз Ғзј–иҜ‘е’Ңе®үиЈ…иҝҷдәӣеҢ…гҖӮдҪҶиҝҷйҖҡеёёйңҖиҰҒиҫғй«ҳзҡ„жҠҖжңҜиғҪеҠӣе’Ңж—¶й—ҙжҠ•е…ҘгҖӮ

жҖ»з»“пјҡ дјҳе…ҲжҹҘжүҫе®ҳж–№еҜ№CondaзҺҜеўғзҡ„ж”ҜжҢҒгҖӮеҰӮжһңе®ҳж–№дёҚж”ҜжҢҒпјҢзӣҙжҺҘдҪҝз”Ёзі»з»ҹPythonзҺҜеўғеҸҜиғҪжҳҜжңҖзЁіеҰҘзҡ„ж–№жЎҲгҖӮйҒҝе…ҚйҖҡиҝҮдҝ®ж”№PYTHONPATHзӯүж–№ејҸејәиЎҢж··еҗҲдёҚеҗҢзҺҜеўғзҡ„еҢ…пјҢйҷӨйқһжӮЁйқһеёёжё…жҘҡжҪңеңЁзҡ„йЈҺйҷ©е’ҢеҰӮдҪ•и§ЈеҶіеҶІзӘҒгҖӮ

Q22: зј–иҜ‘иҮӘе·ұејҖеҸ‘зҡ„LinuxеҶ…ж ёжЁЎеқ—пјҲ.koж–Үд»¶пјүеҗҺпјҢеҠ иҪҪж—¶йҒҮеҲ°вҖңй©ұеҠЁзӯҫеҗҚжҠҘй”ҷвҖқжҲ–вҖңRequired key not availableвҖқзӯүй—®йўҳжҖҺд№ҲеҠһпјҹвҖӢ

A: иҫғж–°зүҲжң¬зҡ„LinuxеҶ…ж ёпјҢе°Өе…¶жҳҜеңЁеҗҜз”ЁдәҶSecure Bootзҡ„зі»з»ҹдёҠпјҢжҲ–иҖ…жҹҗдәӣеҸ‘иЎҢзүҲзҡ„й»ҳи®Өе®үе…Ёзӯ–з•ҘпјҢдјҡиҰҒжұӮеҠ иҪҪеҲ°еҶ…ж ёдёӯзҡ„жЁЎеқ—пјҲ.koж–Үд»¶пјүеҝ…йЎ»е…·жңүжңүж•Ҳзҡ„ж•°еӯ—зӯҫеҗҚгҖӮеҰӮжһңжӮЁиҮӘиЎҢзј–иҜ‘дәҶдёҖдёӘеҶ…ж ёжЁЎеқ—дҪҶжІЎжңүеҜ№е…¶иҝӣиЎҢзӯҫеҗҚпјҢеңЁе°қиҜ•дҪҝз”Ё insmod жҲ– modprobe еҠ иҪҪж—¶е°ұеҸҜиғҪйҒҮеҲ°жӯӨзұ»й”ҷиҜҜгҖӮ

и§ЈеҶіж–№жі•йҖҡеёёж¶үеҸҠд»ҘдёӢжӯҘйӘӨпјҡ

  1. з”ҹжҲҗзӯҫеҗҚеҜҶй’ҘеҜ№пјҡ жӮЁйңҖиҰҒдёҖеҜ№е…¬й’Ҙе’Ңз§Ғй’Ҙз”ЁдәҺзӯҫеҗҚгҖӮеҸҜд»ҘдҪҝз”Ё openssl е·Ҙе…·з”ҹжҲҗгҖӮ

    openssl req -new -x509 -newkey rsa:2048 -keyout MOK.priv -outform DER -out MOK.der -nodes -days 36500 -subj "/CN=My Module Signing Key/"

    иҝҷдјҡз”ҹжҲҗз§Ғй’Ҙ MOK.priv е’Ңе…¬й’Ҙ MOK.derгҖӮ

  2. еҜ№еҶ…ж ёжЁЎеқ—иҝӣиЎҢзӯҫеҗҚпјҡ LinuxеҶ…ж ёжәҗз Ғж ‘дёӯйҖҡеёёеҢ…еҗ«дёҖдёӘзӯҫеҗҚи„ҡжң¬ scripts/sign-fileгҖӮжӮЁйңҖиҰҒдҪҝз”ЁиҝҷдёӘи„ҡжң¬гҖҒжӮЁзҡ„з§Ғй’Ҙд»ҘеҸҠеҲҡз”ҹжҲҗзҡ„е…¬й’ҘпјҲжҲ–иҖ…еҶ…ж ёдҝЎд»»зҡ„жҹҗдёӘе…¬й’ҘпјүжқҘеҜ№зј–иҜ‘еҘҪзҡ„ .ko ж–Үд»¶иҝӣиЎҢзӯҫеҗҚгҖӮ

    # еҒҮи®ҫжӮЁеңЁеҶ…ж ёжәҗз Ғзӣ®еҪ•дёӢпјҢе№¶дё” MOK.priv е’Ң MOK.der еңЁеҪ“еүҚзӣ®еҪ•
    # KBUILD_SIGN_PIN зҺҜеўғеҸҳйҮҸеҸҜиғҪйңҖиҰҒи®ҫзҪ®пјҢз”ЁдәҺиҮӘеҠЁзӯҫеҗҚиҝҮзЁӢдёӯзҡ„еҜҶз ҒдәӨдә’пјҲеҰӮжһңеҜҶй’ҘжңүеҜҶз Ғпјү
    sudo /usr/src/linux-headers-$(uname -r)/scripts/sign-file sha256 ./MOK.priv ./MOK.der /path/to/your/module.ko

    и·Ҝеҫ„е’Ңе…·дҪ“еҸӮж•°еҸҜиғҪйңҖиҰҒж №жҚ®жӮЁзҡ„еҶ…ж ёзүҲжң¬е’ҢзҺҜеўғиҝӣиЎҢи°ғж•ҙгҖӮ

  3. е°ҶзӯҫеҗҚе…¬й’ҘжіЁеҶҢеҲ°зі»з»ҹзҡ„MOK (Machine Owner Key) еҲ—иЎЁдёӯпјҡ дёәдәҶи®©еҶ…ж ёдҝЎд»»жӮЁзҡ„зӯҫеҗҚпјҢйңҖиҰҒе°ҶзӯҫеҗҚж—¶дҪҝз”Ёзҡ„е…¬й’ҘпјҲMOK.derпјүеҜје…ҘеҲ°зі»з»ҹзҡ„MOKеҲ—иЎЁдёӯгҖӮиҝҷйҖҡеёёйҖҡиҝҮ mokutil е·Ҙе…·е®ҢжҲҗгҖӮ

    sudo mokutil --import MOK.der

    жү§иЎҢжӯӨе‘Ҫд»ӨеҗҺпјҢзі»з»ҹдјҡжҸҗзӨәжӮЁи®ҫзҪ®дёҖдёӘдёҙж—¶еҜҶз ҒгҖӮиҜ·и®°дҪҸиҝҷдёӘеҜҶз ҒгҖӮ

  4. йҮҚеҗҜзі»з»ҹе№¶еңЁMOK ManagerдёӯзЎ®и®ӨеҜје…Ҙпјҡ йҮҚеҗҜи®Ўз®—жңәгҖӮеңЁеҗҜеҠЁиҝҮзЁӢдёӯпјҲйҖҡеёёеңЁUEFI/BIOSд№ӢеҗҺпјҢж“ҚдҪңзі»з»ҹеҠ иҪҪд№ӢеүҚпјүпјҢзі»з»ҹдјҡиҝӣе…ҘдёҖдёӘи“қиүІзҡ„MOK Managerз•ҢйқўпјҲShimпјүгҖӮжӮЁйңҖиҰҒйҖүжӢ© "Enroll MOK" жҲ–зұ»дјјйҖүйЎ№пјҢз„¶еҗҺиҫ“е…Ҙд№ӢеүҚи®ҫзҪ®зҡ„дёҙж—¶еҜҶз ҒпјҢжқҘзЎ®и®ӨеҜје…Ҙж–°зҡ„е…¬й’ҘгҖӮ

  5. еҠ иҪҪе·ІзӯҫеҗҚзҡ„жЁЎеқ—пјҡ е®ҢжҲҗд»ҘдёҠжӯҘйӘӨеҗҺпјҢжӮЁзҡ„еҶ…ж ёжЁЎеқ—еә”иҜҘе·Із»Ҹиў«жңүж•ҲзӯҫеҗҚпјҢе№¶дё”еҶ…ж ёдјҡдҝЎд»»иҝҷдёӘзӯҫеҗҚгҖӮжӯӨж—¶еҶҚе°қиҜ•еҠ иҪҪ .ko ж–Үд»¶еә”иҜҘе°ұдёҚдјҡеҶҚжҠҘзӯҫеҗҚй”ҷиҜҜдәҶгҖӮ
